情報処理学会 試行標準 IPSJ-TS 0008:2007


大規模漢字集合の異体字構造 解説




1. 公表の趣旨


この学会試行標準(IPSJ-TS)は，漢字を扱う研究機関及び教育機関における異体字を考慮した情報処理を可能にすることを目的として，ISO/IEC 
10646:2003 
国際符号化文字集合に収録される漢字集合に含まれる多くの異体字関係を，その種類及び特徴に応じて構造的に表現するための新たな調査研究を行い，定めたものである。 

ISO/IEC JTC1/SC2/WG2/IRG(Ideographic Rapporteur 
Group)における国際的な大規模漢字集合の異体字構造の作成に先立ち, 日本国内での合意形成を目的として, IPSJ-TS 0008を公表する。 




2. 背景


日本語の文書を記述する際の漢字使用の目安である常用漢字は1945字であるが，一般の漢和辞典には，約1万字の漢字が収録されている。一方で“康煕字典”,“大漢和辞典”,“漢語大字典”などの大形字典では，5万字前後の漢字が収録されている。そして，情報処理の世界ではISO/IEC 
10646:2003 (Unicode)といった国際規格により，すでに7万字以上の漢字が符号化されているが，その大多数は“異体字”と呼ばれるものである。 


その“異体字”は，それぞれの国や地域において，常用字・表外字（日本），簡化字・繁体字（中国）といったさまざまな関係をもっている。その“異体字”の扱いが資料検索などの面で大きな問題になっている。そこで，大規模な漢字集合における異体字関係の構造を整理，統合し，その成果を具体的に提供するための審議を行った。 




3. 審議中の主要検討課題


審議の検討課題とその過程は次のとおりである。 


3.1 既存の大規模文字集合に含まれる異体字構造の分析


本委員会の初期に，これまでソフトウェアベンダがフォント開発のために制作した文字表に含まれる異体構造の分析を行なったところ，日本漢字は，1946年以降の文字政策によって大幅な漢字の書き換えが行なわれたために，他の国で使用される漢字よりも複雑な異体字構造をもつことが明らかになった。そのため，本委員会における異体字構造の分析には，日本漢字をベースに作業し，その後他の漢字圏の言語への応用することが作業効率上も有効であるとの結論に達した。 



3.2 異体字構造の活用ニーズの確認


博物館・図書館が所蔵する所蔵品のカタログを電子形式で作成する場合，異体字が代表字に置き換えられることがある。しかし，現在の置き換え作業には，二つの点で問題点がある。一つは置き換えが手作業で行なわれているという点である。もう一つはカタログへの入力時に字形を置き換えてしまうため，カタログ上の情報からは元の字形を復元することができない点である。本委員会の異体字構造をこのようなカタログやデータベース作成に活用することで，現行の作業の簡素化や効率化が期待できる。 



3.3 異体関係抽出のアルゴリズムの作成とそれに基づく試表の作成


理想的には，既存の文字表に含まれる異体字構造だけを参照して，異体字表が作成できることが望ましい。本委員会では，複数の字形資料から異体関係をもつ字形を選定するためのアルゴリズムを決定し，異体字構造の試表を作成した。その結果，このような機械的な手法によって作成した異体字表には，別字形として提示すべきではない字形まで含まれることが明らかになった。したがって，字形使い分けの指針を作成する上では，機械的な作業による字形リストを作成した後に，個々の字形がもつ異体関係情報を検討しなおすことが必要であることが確認された。 




4. 懸案事項


2種類の表形式を提示することで，ユーザの目的に沿った情報を提供することが可能であると考えているが，異体字関係データを大規模に収集しているため，一目で異体集合の特徴を把握することは難しい。ユーザ側で自由に応用できることを想定した結果，2種類の表を提示することでその目的が達成できると判断としたが，ユーザにとってより把握の容易な表現形態については検討の余地がある。
